# 霹雳游侠 - K.I.T.T. 界面模拟器

这是一个复古风格的网页模拟器，模拟了经典 80 年代电视剧《霹雳游侠》（Knight Rider）中的奈特工业 2000（K.I.T.T.）界面。

## 功能特点

*   **3D 线框引擎**：一个定制的轻量级 JavaScript 3D 引擎，用于渲染旋转的庞蒂亚克火鸟 Trans Am 线框模型。
*   **动态扫描灯**：标志性的红色扫描灯在 3D 空间中完全动画化，并随车辆旋转而移动。
*   **复古美学**：采用 CRT 扫描线、发光矢量线条和 80 年代计算机终端字体（Orbitron），完美复刻剧中的仪表盘风格。
*   **合成音效**：包含使用 Web Audio API 实现的经典扫描仪低频音效。

## 如何运行

只需在任何现代网络浏览器中打开 `index.html` 文件即可。

```bash
# 如果你安装了 Python，可以运行一个简单的服务器
python -m http.server
# 然后访问 http://localhost:8000
```

## 控制说明

*   **POWER (电源)**：激活系统，启动 3D 渲染循环和扫描仪动画。
*   **AUDIO (音频)**：切换合成的扫描仪音效（需要电源开启）。

## 技术栈

*   HTML5 Canvas
*   CSS3 (Flexbox, 渐变, 动画)
*   原生 JavaScript (未使用外部 3D 库 - 所有矢量数学运算均为手写)
*   Web Audio API
